# 📊 Machine Learning Assignment – Linear Regression

## 📌 Overview
This project demonstrates a simple **Machine Learning workflow using Linear Regression**.  
It includes data analysis, visualization, model building, and evaluation using Python.

---

## 📁 Dataset
- File used: `MBA Salary.csv`
- Feature: Percentage in Grade 10
- Target: Salary (INR)

---

## 🧠 Concepts Covered
- Machine Learning basics  
- Types of ML (Supervised, Unsupervised, Semi-supervised)  
- Feature Scaling  
- Linear Regression  
- Model Evaluation Metrics  

---

## ⚙️ Workflow

### 1. Import Libraries
Used libraries:
- pandas
- numpy
- matplotlib
- seaborn
- sklearn

---

### 2. Load Dataset
Dataset is loaded using pandas and initial exploration is done using:
- head()
- info()
- describe()
- correlation matrix

---

### 3. Exploratory Data Analysis (EDA)
- Checked relationship between features
- Visualized using scatter plot

---

### 4. Train-Test Split
Dataset is split into:
- Training data (60%)
- Testing data (40%)

---

### 5. Model Building
Linear Regression model is trained using sklearn.

---

### 6. Predictions
Model is used to predict salary based on test data.

---

### 7. Model Evaluation
Performance is measured using:
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- Mean Absolute Error (MAE)
- R² Score

---

## 📊 Results
The model successfully learns the relationship between **10th percentage and salary** and provides predictions using a best-fit line.

---

- Python

## 📌 Conclusion
Linear Regression is a simple and powerful algorithm used for predicting continuous values.  
It helps understand relationships between variables and is widely used in real-world applications like salary prediction, sales forecasting, and trend analysis.
